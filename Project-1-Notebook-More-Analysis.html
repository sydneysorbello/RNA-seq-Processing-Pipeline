This is an R Markdown Notebook for Project 1 RNAseq. In this notebook we will first inspect the MultiQC results for quality. Next we can start our analysis by filtering the counts matrix. Then we will perform differential expression analysis and fgsea.

Lets look at the MultiQC Analysis and analyze for overall quality. The Alignment statistics show a very good alignment percentage in which all samples are over 98%. The STAR alignment also reveals an slightly abnormal number of reads for the sample control_rep_3. This sample had a total of 118.9M reads whereas the other samples ranged from 84.5M to 106.9M. Although this is not necessarily a major red flag in the analysis, it is something we should take note of. Another indication that the STAR alignment was largley successful is that only a very small percentage of reads were mapped to too many loci or were un-mapped.

We can also look the the FastQC results for each sample. Since we are looking at RNAseq results, we expect to see a large number of duplicate reads. There is a good mix of single and duplicate reads for each sample, a good for for our results. FastQC also reveals the Sequence Quality Histograms. This feature allows us to look at the mean quality value across each base position in the read. For all of the samples we can see a high seqeunce quality across the reads. All of the samples also passed the per sequence quality score test. The per base sequence content test, however added a warning or failed the samples. This is most likely due to primer sequences that were used in the experiment, it does not indicate poor quality for an RNAseq experiment. Looking at the GC content we see roughly 50% on each of the samples; aligned with what is expected.

When looking at the top over-represented sequences, we can see they each have reads in the hundreds of thousands. The top over-represented sequence: ATCGGAAGAGCACACGTCTGAACTCCAGTCACTAATACAGATCTCGTATG was put into BLAST. Although it wasnt 100% identical to any gene, it shared 94.12% identity with a severe acute respiratory syndrome in Coronavirus 2 genome assembly. This may be an interesting find, but given every other quality measure looks ok, we can continue with analysis.

As we can see theres a some reads that have a count of zero for each sample. In order to filter the counts, we will remove genes that have zero counts.

filtered_counts <- raw_counts %>%
  filter(
    rowSums(across(-1)) != 0,
    rowMeans(across(-1)) >= 0.5
  )

head(filtered_counts)

In order to perform differential expression analysis with DESeq2, we need to correctly format the inputs:

count_matrix <- filtered_counts %>%
  column_to_rownames("gene") %>%
  as.matrix()

sample_conditions <- data.frame(
  row.names = colnames(count_matrix),
  condition = c("control", "control", "experiment", "experiment", "experiment", "control")
)

Now lets create the dataset for DESeq2:

dds <- DESeqDataSetFromMatrix(
  countData = count_matrix,
  colData = sample_conditions,
  design = ~ condition
)

Now that we have performed differential expression, lets visualize the results. First well make a volcano plot:

res_df <- as.data.frame(res) %>%
  rownames_to_column("gene") %>%
  mutate(
    neg_log10_padj = -log10(padj),
    significance = case_when(
      padj < 0.05 & log2FoldChange > 1 ~ "Up",
      padj < 0.05 & log2FoldChange < -1 ~ "Down",
      TRUE ~ "NotSig"
    )
  )

# Show the 10 most significant gene names:
top_genes <- res_df %>%
  filter(significance != "NotSig") %>%
  arrange(padj) %>%
  slice_head(n = 10)

# and finally, plot the volcano plot:
ggplot(res_df, aes(x = log2FoldChange, y = neg_log10_padj)) +
  geom_point(aes(color = significance), alpha = 0.8) +
  scale_color_manual(values = c("Up" = "brown1", "Down" = "cyan3", "NotSig" = "black")) +
  geom_vline(xintercept = c(-1, 1), linetype = "dotted", color = "darkgray") +
  geom_hline(yintercept = -log10(0.05), linetype = "dotted", color = "darkgray") +
  geom_text(
    data = top_genes,
    aes(label = gene),
    vjust = -1, size = 3.5
  ) +
  labs(
    x = "Expression difference (log2FC)",
    y = "Significance (-log10 p value)",
    title = "Volcano Plot"
  ) +
  theme_minimal(base_size = 14)
